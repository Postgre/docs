Normes de développement

Les scripts

Règles de nommage

 

entêtes

 

Tout script doit commencer par un entête.

Vous devrez utiliser ce cartouche pour les scripts :

 

-- ---------------------------------------------------------------------------

-- script : <nom du script>

-- objet :  <utilité du script>

--

-- desc :   <description du script et de son utilisation>

-- ---------------------------------------------------------------------------

-- Auteur:                                     Date de création:

-- ---------------------------------------------------------------------------

-- Auteur:                                     Date de modif.:

-- modifs:

-- ---------------------------------------------------------------------------

 

Exemple d’utilisation :

 

-- ---------------------------------------------------------------------------

-- script : gen_cre_syn_pub.sql

-- objet :  script de génération des synonymes public

--

-- desc :   Lancer ce script pour générer un script qui va créer les synonymes

--          publics des tables appartenant à l'utilisateur connecté.

-- ---------------------------------------------------------------------------

-- Auteur: C.Domas                             Date de création: 09/12/1999

-- ---------------------------------------------------------------------------

-- Auteur:                                     Date de modif.:

-- modifs:

-- ---------------------------------------------------------------------------

 

Afin de faciliter le travail de vos successeurs, pensez toujours à laisser une ligne permettant d’ajouter les commentaires sur les futures modifications :

 

-- ---------------------------------------------------------------------------

-- Auteur: C.Domas                             Date de création: 09/12/1999

-- ---------------------------------------------------------------------------

-- Auteur: C.Domas                             Date de modif.:   11/12/1999

-- modifs: ajout d’un log pour l’exécution du script

-- ---------------------------------------------------------------------------

-- Auteur:                                     Date de modif.:

-- modifs:

-- ---------------------------------------------------------------------------
