Aide mémoire Commandes unix

cat fichier

affiche le contenu de fichier

more fichier

affiche le contenu de fichier page par page

head -n nombre fichier

affiche les nombre premières lignes de fichier

tail -n nombre fichier

affiche les nombre dernières lignes de fichier

cp fichier1 fichier2

copie fichier1 dans fichier2

mv fichier1 fichier2

renomme fichier1 en fichier2

ln fichier1 fichier2

crée un lien (fichier2) de fichier1. Il s'agit physiquement du même fichier.

ln -s fichier1 fichier2

crée un lien symbolique (fichier2) de fichier1.

rm fichier

efface fichier

ls -l

affiche le contenu du répertoire courant sous format long

-rw-r--r-- 1 babafou skiff 94 Aug 11 18:02 toto

 

Les différentes informations sont les suivantes :

-rw-r--r-- type et droits d'accès du fichier

 

Le premier caractère représente le type du fichier :

-

fichier normal

d

répertoire

l

lien symbolique

b

fichier spécial de type bloc

c

fichier spécial de type caractère

s

socket

Les suivants représentent les droits d'accès au fichier et se lisent par groupe de trois : le premier concerne le propriétaire du fichier, le second son groupe et le troisième les autres utilisateurs.
Ces droits sont de trois types :

r

droit en lecture (possibilité de faire ls pour un répertoire)

w

droit en écriture (possibilité de créer et d'effacer des fichiers pour un répertoire)

x

droit en exécution (possibilité de faire cd pour un répertoire)

Un - à la place d'un droit indique que ce droit est refusé.
Ainsi, dans notre exemple, le fichier toto est accessible en lecture et en écriture (rw-) à l'utilisateur babafou, en lecture (r--) au groupe skiff et en lecture (r--) pour tous les autres utilisateurs.

1

nombre de liens sur le fichier

babafou

propriétaire du fichier

skiff

groupe du fichier

94

taille du fichier

Aug 11 18:02

date de dernière modification du fichier

toto

nom du fichier

chmod mode fichier

modifie les droits d'accès à fichier selon la valeur de mode. Ce mode peut être spécifié de deux manières différentes :

en octal

1

au droit d'exécution

2

au droit d'écriture

4

au droit de lecture

et, pour chaque permission (propriétaire, groupe, autres), on fait la somme des droits qui lui sont accordés. Ainsi,

rw-r--r-- s'écrit 644 et

rwxr-xr-x s'écrit 755.

 

sous forme symbolique

On précise les droits à appliquer sous la forme d'une chaîne de caractères composée de trois parties :

la première indique pour qui on va modifier les droits d'accès :

u

le propriétaire du fichier (user)

g

le groupe du fichier (group)

o

les autres (others)

a

tout le monde (all), c'est équivalent à ugo

la seconde indique s'il faut

+

ajouter des droits

-

supprimer des droits

=

fixer des droits

la troisième indique les droits à appliquer :

r

lecture

w

écriture

x

exécution

Ainsi, la commande chmod go-rw toto supprime les droits de lecture et d'écriture pour le groupe et les autres sur le fichier toto.

 

mkdir répertoire

crée le répertoire répertoire

rmdir répertoire

efface le répertoire vide répertoire

cp -R répertoire1 répertoire2

copie le répertoire répertoire1 et tout son contenu sous le nom répertoire2

rm -r répertoire

efface le répertoire répertoire et tout ce qu'il contient
